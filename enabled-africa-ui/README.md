# Enabled Africa

Enabled Africa is a single-page React application for talent profiles, employer tools, and community services — connecting people and opportunities across the continent.

The application uses role-based routes for students, employers, and service providers.

---

## Tech Stack

- React
- JavaScript (ES6+)
- CSS (custom styling)
- Create React App

---

## Prerequisites

- Node.js (LTS recommended)  
  https://nodejs.org

Verify installation:

```bash
node -v
npm -v
```

```bash
git clone <your-repository-url>
cd enabled-africa-ui
npm install
```

## Theme
